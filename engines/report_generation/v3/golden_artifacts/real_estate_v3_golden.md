## 4.5 术语表

| 术语 | 解释 |
|------|------|
| 定金 | 合同当事人约定的担保形式，收受定金方违约应双倍返还，支付方违约无权要求返还 |
| 订金 | 预付款性质，不适用双倍返还罚则，仅需原额返还 |
| 过户 | 不动产物权变更登记，房屋所有权转移的法定要件 |
| 继续履行 | 守约方有权要求违约方按合同约定继续履行合同义务 |
| 违约金酌减 | 约定违约金过高时，法院可依当事人请求酌情调整至实际损失水平 |
| 抵押权 | 债务人或第三人以不动产为债权提供担保的物权，未解除前影响过户 |
| 重大误解 | 行为人对行为性质、对方当事人、标的物等产生错误认识的可撤销事由 |
